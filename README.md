#ScreenshotUploaderPL
*by Nicolapcweek94*

Another version of my screenshot uploader, this time in Perl with SHA3 hashes. 

##Requirements:
 + Net::FTPSSL
 + Digest::SHA3
 + Clipboard
 + Perl version >= 5.10
